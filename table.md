# Table System (Item Detection)

`br-table` は、RFID を使用してアイテムの使用を検知するためのハードウェアコンポーネントです。Arduino または ESP32 で制御される予定です。

## 概要

プレイヤーがテーブル上の所定の位置（リーダー上）にアイテム（RFIDタグ付き）を置くことで、そのアイテムの使用をシステムに通知します。

## ハードウェア構成 (予定)

- **マイコン**: Arduino Uno / Nano または ESP32
- **RFIDリーダー**: MFRC522 (SPI接続)
- **通信**:
    - ESP32の場合: Wi-Fi (TCP/UDP)
    - Arduinoの場合: USBシリアル通信 (Master PCに直接接続)

## 機能

1. **タグ検知**: RFIDタグのUIDを読み取ります。
2. **データ送信**: 読み取ったUIDをMasterサーバーに送信します。

## 通信プロトコル

Master サーバーとの通信仕様（案）。

### 送信データ (Table -> Master)

- アイテム検知: `RFID:<UID>`
    - `<UID>`: タグの固有ID (例: `A1:B2:C3:D4`)

Master側では、このUIDとアイテム種別のマッピングテーブルを持っており、どのアイテムが使用されたかを判定します。
